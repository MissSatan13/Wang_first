学生成绩管理系统
================

功能介绍
--------
1. 学生信息管理
   - 添加学生信息
   - 修改学生信息
   - 删除学生信息
   - 查询学生信息

2. 成绩管理
   - 录入学生成绩
   - 修改学生成绩
   - 删除学生成绩
   - 查询学生成绩

3. 统计分析
   - 计算学生平均分
   - 计算班级平均分
   - 成绩排名

使用说明
--------
1. 运行环境要求：
   - Python 3.6或以上版本
   - 操作系统：Windows/Linux/MacOS

2. 运行方式：
   - 直接运行main.py文件
   - 按照菜单提示进行操作

3. 数据存储：
   - 学生信息存储在students.txt文件中
   - 成绩信息存储在scores.txt文件中
